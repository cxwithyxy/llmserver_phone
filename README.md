## llmserver_phone
该项目旨在手机端启动大语言模型服务，使得其他自动化软件（macrodroid、tasker）可以利用http服务调用大语言模型来处理。

主要依赖于 google-ai-edge/gallery 这个开源项目，进行了二次修改

---

## 增加了web server，可以基于http服务让大语言模型被访问并回传数据。

接口url
```
http://127.0.0.1:8081/chat
```

post数据如下
```
{"message":"你好呀，我已今天心情不错，你呢","conversation_id":"test-session-1","model": "DeepSeek-R1-Distill-Qwen-1.5B"}
```

回传数据如下
```
{ "success": true, "model": "DeepSeek-R1-Distill-Qwen-1.5B", "conversation_id": "test-session-1", "response": "好的，用户说“你好呀，我已今天心情不错，你呢”，看起来是在打招呼，然后补充说“你好呀，我今天心情不错，你呢”。可能是在确认对方今天心情不错，或者是在闲聊。 首先，我需要理解用户的需求。他可能是在打招呼，然后想确认对方的天气情况，或者是在闲聊，可能想表达对他的兴趣或者关心。 接下来，我要考虑如何回应。因为用户已经确认了天气不错，所以可以继续友好地回应，比如“你好呀，今天心情不错，谢谢你的关心！有什么我可以帮你的吗？”这样既回应了问候，又表达了关心，同时邀请对方进一步交流。 另外，用户可能希望对方感受到被重视和关心，所以语气要自然、亲切，避免过于生硬或夸张。 最后，我要确保回应简洁明了，同时包含足够的信息，让对方知道我愿意帮助，同时也能保持轻松的语气，让对方感到被重视。 \u003c/think\u003e 你好呀，今天心情不错，谢谢你的关心！有什么我可以帮你的吗？", "latency_ms": 11688.0 }
```
